在 VM 上安裝 macOS Monterey

發表於 2021-12-08 分類於 技術
文章字數： 791 所需閱讀時間 ≈ 1 分鐘

目的

說明在虛擬機上，安裝 macOS Monterey 的步驟。

前言

最近在使用 IONIC 開發 APP應用，其中 IOS 版本，得需要 Mac OS 才能編譯、發佈、上架到APP Stroe，因為資源有限，所以利用虛擬機安裝 Mac OS 的開發環境。

開發環境與版本

  1. 作業系統：Windows 10
  2. 虛擬軟體：VMware Workstation Pro 14
  3. 作業系統：macOS Monterey

筆記

安裝 macOS Monterey

  1. 選擇語系。
  2. 選擇【磁碟工具程式】，並點選【繼續】。
  3. 由於磁碟尚未初始化，先由左列選擇要清除的磁碟，並點選【清除】。
  4. 更改名稱，並點選【清除】。
  5. 清除和製作完成後，點選【完成】。
  6. 左上角點選【磁碟工具程式】，並結束【磁碟工具程式】。
  7. 選擇【安裝 macOSA Monerey】，並點選【繼續】。
  8. 點選【繼續】。
  9. 點選【同意】。
  10. 點選【同意】。
  11. 選擇要安裝的【磁碟】，點選【繼續】。
  12. 等待安裝中。
  13. 安裝完成後，選擇國家或地區，點選【繼續】。
  14. 點選【繼續】。
  15. 點選【稍後再說】。
  16. 點選【繼續】。
  17. 點選【稍後再說】。
  18. 如果有 Apple ID，可使用Apple ID 登入。這邊選擇，【稍後設定】。
  19. 點選【略過】。
  20. 點選【同意】。
  21. 點選【同意】。
  22. 建立電腦帳號，並點選【繼續】。
  23. 點選【繼續】。
  24. 點選【不使用】。因為是虛擬機開發，用不到定位，要選擇使用，也可以。
  25. 選擇【時區】，點選【繼續】。
  26. 點選【繼續】。
  27. 點選【稍後設定】。
  28. 選擇【外觀】，點選【繼續】。

安裝 VMware Tools

  1. 選擇工具列【VM】 => 【Install VMware Tools…】
  2. 選擇【安裝 VMware Tools】
  3. 點選【繼續】。
  4. 點選【安裝】。
  5. 輸入對應的帳號密碼，並點選【安裝軟體】。
  6. 點選【好】。
  7. 點選【打開安全性偏好設定】。
  8. 左下角，點選鎖頭圖示，以進行修改。
  9. 輸入對應的帳號密碼，並點選【解鎖】。
  10. 點選【允許】。
  11. 點選【稍後再說】，最後再選擇關機，這一步先不關機。
  12. 關閉視窗。
  13. 安裝完成，點選【重新開機】
